# איך לראות לוגים ב-Vercel

## דרך 1: דרך Deployments (הכי פשוט)

1. **לך ל-Vercel Dashboard** → הפרויקט שלך
2. **לך ל-Deployments** (בתפריט העליון)
3. **לחץ על ה-Deployment האחרון** (הכי למעלה)
4. **לחץ על "Functions"** או "View Function Logs"
5. **חפש את `/api/cron/daily-report`** ברשימה
6. **לחץ עליו** → תראה את כל הלוגים

## דרך 2: דרך Logs (אם יש)

1. **לך ל-Vercel Dashboard** → הפרויקט שלך
2. **לך ל-Logs** (בתפריט העליון)
3. **חפש** את `/api/cron/daily-report` או `daily-report`
4. **לחץ על השורה** → תראה את הפרטים

## דרך 3: דרך Settings → Functions

1. **לך ל-Vercel Dashboard** → הפרויקט שלך
2. **לך ל-Settings** → **Functions**
3. **חפש** את `/api/cron/daily-report`
4. **לחץ עליו** → תראה את הלוגים

## מה לחפש בלוגים:

1. **"Daily report email sent successfully to..."** - כאן תראה לאילו מיילים נשלח
2. **"Found X orders for today"** - כמה הזמנות נמצאו
3. **שגיאות** - אם יש שגיאות, תראה אותן שם

## אם לא מוצא:

1. **נסה לשלוח מייל ידנית** (הקוד שכבר הרצת)
2. **לך מיד ל-Logs** אחרי השליחה
3. **חפש את הזמן הנוכחי** בלוגים
4. **תראה את הלוגים של השליחה**

## טיפ:

אם אתה לא מוצא, נסה:
- **רענן את הדף** (F5)
- **חפש** את המילה "daily-report" או "email"
- **בדוק את הזמן** - הלוגים מסודרים לפי זמן
