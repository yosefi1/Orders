# בדיקה ידנית - שליחת מייל

## הרץ את זה בדפדפן (Console):

```javascript
fetch('https://orders-altera.vercel.app/api/cron/daily-report', {
  headers: {
    'Authorization': 'Bearer altera1'
  }
})
.then(r => r.json())
.then(data => {
  console.log('תוצאה:', data);
  alert('תוצאה: ' + JSON.stringify(data, null, 2));
})
.catch(error => {
  console.error('שגיאה:', error);
  alert('שגיאה: ' + error.message);
});
```

## מה לחפש בתשובה:

### אם הכל עובד:
```json
{
  "success": true,
  "message": "Daily report sent with X orders"
}
```

### אם אין הזמנות להיום:
```json
{
  "message": "No orders for today"
}
```

### אם יש שגיאה:
```json
{
  "error": "שגיאה כלשהי"
}
```

## הערה חשובה:

אם אתה רואה `"No orders for today"` - זה אומר שהקוד עובד, אבל אין הזמנות להיום, אז המייל לא נשלח (זה התנהגות תקינה).

אם אתה רואה `"success": true` - המייל נשלח בהצלחה!
