# איך לשנות את המייל של הספק

המייל של הספק מוגדר במשתנה סביבה `SUPPLIER_EMAIL`. זה המייל שאליו נשלחים הדוחות היומיים.

**✨ תמיכה בכמה נמענים:** אפשר להגדיר כמה כתובות מייל מופרדות בפסיקים, והדוחות יישלחו לכולן!

## שינוי מקומי (פיתוח):

1. פתח את קובץ `.env.local` בתיקיית הפרויקט
2. מצא את השורה:
   ```env
   SUPPLIER_EMAIL=supplier@example.com
   ```
3. שנה את המייל למייל הרצוי:
   
   **מייל אחד:**
   ```env
   SUPPLIER_EMAIL=your-new-supplier@email.com
   ```
   
   **כמה נמענים (מופרדים בפסיקים):**
   ```env
   SUPPLIER_EMAIL=supplier1@email.com, supplier2@email.com, supplier3@email.com
   ```
   
   **הערה:** אפשר להוסיף רווחים אחרי הפסיקים, הם יוסרו אוטומטית.
4. שמור את הקובץ
5. הפעל מחדש את שרת הפיתוח (`npm run dev`)

## שינוי ב-Vercel (ייצור):

1. לך ל-[Vercel Dashboard](https://vercel.com/dashboard)
2. בחר את הפרויקט שלך
3. לך ל-**Settings** → **Environment Variables**
4. מצא את המשתנה `SUPPLIER_EMAIL`
5. לחץ עליו וערוך את הערך:
   - **מייל אחד:** `supplier@email.com`
   - **כמה נמענים:** `supplier1@email.com, supplier2@email.com, supplier3@email.com`
6. לחץ **Save**
7. **חשוב:** צריך לפרוס מחדש את האפליקציה כדי שהשינוי ייכנס לתוקף:
   - לך ל-**Deployments**
   - לחץ על ה-3 נקודות (⋯) ליד ה-deployment האחרון
   - בחר **Redeploy**

## הערות:

- המייל החדש יקבל את הדוחות היומיים החל מה-deployment הבא
- אם אתה רוצה לבדוק מיד, תוכל לשלוח בקשה ידנית ל-endpoint:
  ```bash
  curl -H "Authorization: Bearer YOUR_CRON_SECRET" https://your-app.vercel.app/api/cron/daily-report
  ```
